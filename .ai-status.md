## Design system (established)

Raw CSS is **forbidden**; all UI must use Tailwind CSS semantic tokens and Shadcn/UI primitives only.

### Design tokens (tailwind + globals.css)

| Token | Mapping | Use |
| ----- | ------- | --- |
| `background` | `hsl(var(--background))` → `@theme` / oklch | Primary canvas |
| `foreground` | `hsl(var(--foreground))` → `@theme` / oklch | High-contrast text |
| `primary` | Subdued financial blue (oklch) | Buttons, links |
| `destructive` | `#EF4444` | Negative balances, liability types, admin deletes |
| `asset-alternativa` | `#3B82F6` | Category badges (e.g. Alternativa) |

Defined in `app/globals.css` (`:root`, `.dark`, `@theme inline`). Tailwind v4; no `tailwind.config.ts` (theme in CSS).

## App Router structure (route groups)

Authentication flows are isolated from the core app via Next.js Route Groups. Skeleton pages exist for every required route; each page renders only a single `<h1>[Page Name]</h1>` for routing verification.

### Visual file tree (route group separation)

```
app/
├── layout.tsx              # Root layout (TooltipProvider, fonts, globals.css)
├── page.tsx                # / (home)
├── globals.css
├── favicon.ico
│
├── (auth)/                  # Route group: public auth (no segment in URL)
│   ├── layout.tsx           # Wraps {children} only
│   ├── login/
│   │   └── page.tsx         # /login
│   ├── signup/
│   │   └── page.tsx         # /signup
│   └── forgot-password/
│       └── page.tsx         # /forgot-password
│
└── (app)/                   # Route group: authenticated platform
    ├── layout.tsx           # Wraps {children} only
    ├── dashboard/
    │   └── page.tsx         # /dashboard
    ├── assets/
    │   └── page.tsx         # /assets
    ├── liabilities/
    │   └── page.tsx         # /liabilities
    ├── transactions/
    │   └── page.tsx         # /transactions
    ├── documents/
    │   └── page.tsx         # /documents
    ├── entities/
    │   └── page.tsx         # /entities
    └── settings/
        └── page.tsx         # /settings
```
